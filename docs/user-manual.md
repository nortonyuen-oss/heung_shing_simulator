# 香城模擬器玩家手冊：特殊建築解鎖規格

本頁記錄需要額外城市條件才可興建，或只會在合資格分區自然生成的建築。所有手動建築仍然需要足夠資金、完整空置 footprint 及可建造地面。

## 分區自然生成模型

| 模型級別 | 出現條件 | 機制 |
| --- | --- | --- |
| 科學園 | 高等教育指數至少 0.80；工業建築 footprint 至少 2×2 | 可由新工業發展或現有工業轉型；「科研發展法」提高機率，但不是硬性條件 |
| H 富人住宅 | 綜合質素 0.55、地價 0.50、環境 0.45、健康 0.45、經濟 0.45 | 所有住宅密度均可能出現，受附近 H／UH 密度抑制 |
| UH 富豪住宅 | 只限低密度住宅；綜合質素 0.76、地價 0.72、景觀 0.55、環境 0.68、健康 0.62、經濟 0.65、污染不高於 0.20 | 最高質素區間的基本權重為 5% |
| H 甲級寫字樓 | 綜合質素 0.56、地價 0.50、環境 0.40、經濟 0.52 | CBD、主幹道及大型交通節點會提高機會，受附近 H／UH 密度抑制 |
| UH 世界級摩天大樓 | 只限高密度商業；綜合質素 0.78、地價 0.70、景觀 0.35、環境 0.55、經濟 0.72、污染不高於 0.25；城市同時有股票交易所及機場 | 靠近股票交易所或機場會提高評分；最高質素區間的基本權重為 5% |

L／M 住宅與商業模型沒有額外硬性門檻，按地段質素及權重生成。H／UH 是逐個地段評估，因此沒有一個代表「全城解鎖」的走馬燈時刻。

## 解鎖通知

城市第一次達成一項離散解鎖條件時，走馬燈及香城討論區會刊登一次通知。通知涵蓋人口、城市吸引力、法案、議會項目、高等教育、立法會、股票交易所及科學園解鎖。通知狀態會寫入存檔，避免每月重複播放；已經存在的建築亦會自動視為已通知。
